# MySQL 安装

如下我们有一组这样的数据

| Yunya | Male | 18   | Freelance |      |      |
| ----- | ---- | ---- | --------- | ---- | ---- |
|       |      |      |           |      |      |

粗略一看并不知道它的意思，所以我们需要有**字段**的标注

| 字段 | NAME  | GENDER | AGE  | PROFESSION |
| ---- | ----- | ------ | ---- | ---------- |
| 记录 | Yunya | Male   | 18   | Freelance  |

这样的一组数据在一张数据表中被称之为**记录**，记录是由多个字段信息所构成的。
